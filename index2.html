WHAT'S THAT?

What's That? is a scanner application that allows users to scan food, cleaning, and personal products by barcode in order to get more details about the ingredients in the items that they consume/use on a daily basis.
Built using HTML, CSS, Javascript, AngularJS and Swift.
What's That? won 2nd place overall and the Code4Impact award at the AngelHack Hackathon Seattle.

CHILL

Chill is website that pairs a song with a looped video to create a mood-environment. The song stays the same, but the video can be changed using the arrow on the right of the page. I applied my minimalist design skills when creating this project.
This page was built using HTML, CSS, Javascript, BigVideo.js, and Foundation6.

WATER WISE

Water Wise is an application that provides easier access to water quality reports and aims to raise the awareness of the many chemicals that our water may contain.

CODEPREP

CODEPREP is just what you need to prepare and practice for the high stress dynamic of technical interviews. With our variety of sample questions, varying difficulty and integrated peer review elements, you are able to simulate white-boarding with live feedback.
Built using HTML, CSS, Ruby On Rails, Javascript, jQuery, AJAX, Bootstrap, and Materialize

WORD SCRAMBLE

Word Scramble is a web-based game built using HTML, CSS, jQuery, and Javascript.
This is a simple word scramble game for 1 player. The player clicks on a tile to populate a line of scrambled letters, then the player must then proceed to unscramble the letters, and guess the word.
